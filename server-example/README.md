# Reference servers

Self-contained example implementations of the CarMenu protocol.

| Language | Path | Notes |
|----------|------|-------|
| Python (Flask) | [`python/`](python/) | ~150 lines. Easiest starting point. |

The protocol itself is documented in
[`docs/PROTOCOL.txt`](../docs/PROTOCOL.txt) (and bundled in the APK).

If you write a server in another language, PRs adding more examples
here are welcome.
